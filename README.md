## Run Locally from system

Clone the project

```bash
  git clone https://github.com/pawitpr/pawito-bot.git
```

Go to the project directory

```bash
  cd pawito-bot
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
